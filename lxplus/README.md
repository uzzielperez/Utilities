## Miscellaneous

Starship installation
```bash
curl -sS https://starship.rs/install.sh | sh -s -- --bin-dir ~/.local/bin
```

```
echo 'eval "$(starship init bash)"' >> ~/.bashrc
source ~/.bashrc
```

And then you'll have a nice terminal! 
```bash
ciperez in 🌐 lxplus919 in ~ via 🐍 v3.12.9
```
